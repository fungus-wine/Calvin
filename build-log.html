2026-03-01

Project start

Project kicked off. Defined the three-system architecture — Instinctus (Arduino GIGA R1 firmware), Cogitator (Jetson Orin Nano services), and Explorator (Electron desktop app) — along with the InstinctusKit shared library in the librarius monorepo.

Hardware selection finalized: ODrive S1 motor controllers, D5312S brushless outrunner motors, ICM-20948 9-axis IMU, dual VL53L4CX ToF sensors, and OAK-D Pro W stereo camera. CAN bus at 250 kbps chosen for the motor control link.

This documentation site created. GitHub repositories initialized for all four components.
