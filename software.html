Software Architecture

Calvin's software is split across three purpose-built systems that communicate in a strict hierarchy: Explorator → Cogitator → Instinctus.

01 Instinctus

github.com/fungus-wine/calvin_instinctus

Core M7 – Communications Hub

The M7 core runs at 480 MHz and owns all I/O peripherals. It is the only core that communicates directly with Cogitator and the outside world.

  • Reads ICM-20948 IMU via I²C
  • Polls VL53L4CX ToF sensors
  • Receives commands from Cogitator over UART
  • Publishes sensor data to M4 via event queue
  • Forwards M4 telemetry back to Cogitator

Core M4 – Real-Time Balance Loop

The M4 core runs the hard real-time balance control loop. It consumes IMU-derived tilt angle events from M7 and produces motor velocity commands.

  • Consumes tilt angle from M7 event queue
  • Runs PID controller (~1 kHz target loop rate)
  • Sends velocity setpoints to ODrive S1 over CAN
  • Publishes motor state telemetry to M7

02 InstinctusKit

github.com/fungus-wine/librarius  ·  Shared Arduino library in the librarius monorepo

InstinctusKit is the shared Arduino library that powers inter-core communication between the M7 and M4 cores of the GIGA R1. It provides a typed event queue backed by shared SRAM, with producer/consumer APIs that require no RTOS.

  • Typed event structs (sensor data, commands, telemetry)
  • Lock-free ring buffer in shared SRAM region
  • Header-only Arduino library for easy inclusion
  • Versioned alongside Instinctus in the librarius monorepo

03 Cogitator

github.com/fungus-wine/calvin_cogitator

Planned Python Services

Cogitator is architected as a set of independent Python services coordinated by a lightweight message bus.

  • serial_bridge
    UART link to Instinctus — reads telemetry, writes commands
  • vision_service
    OAK-D depth perception, object detection via DepthAI
  • llm_service (planned)
    Natural-language command processing via local LLM
  • api_gateway
    WebSocket server — exposes telemetry and commands to Explorator

04 Explorator

github.com/fungus-wine/calvin_explorator

Electron + Vue 3

Explorator is a cross-platform desktop application built with Electron and Vue 3. It connects to Cogitator's WebSocket API over LAN to provide live telemetry and manual control.

  • Real-time telemetry dashboard (tilt, speed, distance)
  • Live camera feeds from Cogitator
  • Manual drive controls (keyboard / gamepad planned)
  • PID tuning interface
  • Connection status + reconnection logic
